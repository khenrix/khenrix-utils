# Cross-CLI Parity + Reproducible-Environment Docs — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Inventory every Claude Code plugin/skill/MCP, mirror the feasible set onto Codex + agy, and ship a reproducible-on-a-new-machine setup (inventory manifest + generated doc, idempotent bootstrap, auth/secrets checklist) — all verified by hermetic `--self-test`s plus a documented on-machine smoke.

**Architecture:** One authored desired-state manifest (`docs/environment/inventory.toml`) is the single source of truth; `scripts/env_inventory.py` renders `inventory.md` from it, probes live state into a gitignored report, and `--check`s live-vs-desired. `scripts/bootstrap-machine.sh` reproduces the capability set idempotently (check-before-act, `--dry-run`), using a stdlib `scripts/lib/mcp_merge.py` helper for agy's JSON config. `docs/environment/auth-and-secrets.md` extends the existing `docs/machine-setup.md` with the cross-CLI auth/secrets checklist. Every deterministic piece is TDD'd via the repo's `--self-test` convention wired into `make eval-test`; runtime install/parity steps are executed on the machine and verified by `env_inventory.py --check` + an MCP/skill smoke.

**Tech Stack:** Python 3.11+ **stdlib only** (`tomllib`, `json`, `subprocess`, `re`, `pathlib`, `argparse`, `urllib.parse`, `shutil`, `os`), Bash, GNU make. No pip dependencies.

**Design spec:** `docs/superpowers/specs/2026-07-12-cross-cli-parity-and-env-docs-design.md` (council-converged, commit `bcd57d7`). Read it before starting.

## Global Constraints

- **Python is stdlib-only** — no third-party deps; must run on any Python 3.11+ with no install step (repo `CLAUDE.md`).
- **Tests are `--self-test` subcommands** wired into `make eval-test` — the repo's convention (see `Makefile:64-70`). Not pytest.
- **Never emit a secret value.** The no-secrets guarantee is default-on recursive sanitization; `checks.py` is a shape-limited backstop only.
- **`make verify` must stay green** and **no skill receipts are touched** — this effort adds docs + scripts, not skills. Do not add any hard gate to `make verify`.
- **Never edit `marketplaces/**`** (generated by `render.py`). This effort touches none of it.
- **Skip wikisync** — do not touch `shared/lib/wikisync/**` or the `claude-obsidian` wiki skills; a concurrent workstream owns them.
- **Push is HELD** — commit locally to `main` (solo repo), never `git push`, until the user says the checkout is settled.
- **Scope every `git add` to named paths** — never `git add -A` / `git add .` (a concurrent session may have uncommitted files). Verify `git diff --cached --name-only` before each commit.
- **Repro target is the capability set, not exact versions** — record source + resolved version as best-effort; no checksum/commit-pin lock manifest.

## File Structure

- **Create** `docs/environment/inventory.toml` — authored desired-state manifest (reproduction target). Owns: the canonical list of plugins/MCPs, their per-CLI intended status, XOR/ownership flags.
- **Create** `docs/environment/inventory.md` — **generated** by `env_inventory.py --render`; committed; never hand-edited.
- **Create** `scripts/env_inventory.py` — D1 engine. Owns: manifest load+schema-validate, recursive sanitizer, `--render` (toml→md), live probes (JSON-preferred / config-file fallback), `--check` (live-vs-desired classifier + XOR assertion), the D3 doc completeness check, and `--self-test`.
- **Create** `scripts/lib/mcp_merge.py` — stdlib agy `mcp_config.json` merge helper. Owns: atomic merge with defined conflict semantics + `--self-test`.
- **Modify** `scripts/lib/checks.py` — add `scan_path(path)` file-scoped secret shim (guarded if-exists) + a self-test case.
- **Create** `scripts/bootstrap-machine.sh` — idempotent per-CLI reproducer with `--dry-run`. Owns: prereqs + per-CLI install/register steps (check-before-act).
- **Create** `docs/environment/auth-and-secrets.md` — D3 checklist; cross-links and extends `docs/machine-setup.md` (does not duplicate it).
- **Modify** `Makefile` — add `env_inventory.py --self-test` and `mcp_merge.py --self-test` to the `eval-test` target.
- **Modify** `.gitignore` — ignore the gitignored observed-state report path.

**Ordering rationale:** the manifest (T1) is the contract everything else reads; `env_inventory.py` is built capability-by-capability (T2 schema → T3 sanitize → T4 render → T5 probe → T6 check → T10 doc-completeness) so each step is independently testable; `checks.py` shim (T7) and `mcp_merge.py` (T8) are standalone helpers; `bootstrap-machine.sh` (T9) consumes `mcp_merge`; D3 doc (T10) and the on-machine parity/smoke runbook (T11) come last.

**Note on probe code (spec-mandated discovery, NOT a placeholder):** the spec requires *confirming each `<cli> plugin/mcp list [--json]` subcommand exists and is parseable before committing to it*. Task 5 therefore begins by capturing real output on this machine, saving it as a fixture, and implementing the parser against that captured fixture. The plan gives the parser scaffold + the fixture-driven test contract; the exact field extraction is finalized against the captured shape at execution time. This is a deliberate discovery step, not deferred work.

#!/usr/bin/env python3
"""env_inventory.py — cross-CLI environment inventory (D1).

Reads the desired-state manifest (docs/environment/inventory.toml), renders
inventory.md from it, probes live CLI state into a gitignored report, and
--checks live-vs-desired. Read-only. Stdlib-only. Tests: `--self-test`.
"""
from __future__ import annotations
import argparse, json, os, re, subprocess, sys, tomllib
from pathlib import Path
from urllib.parse import urlsplit, urlunsplit

ROOT = Path(__file__).resolve().parent.parent
MANIFEST = ROOT / "docs/environment/inventory.toml"
DOC = ROOT / "docs/environment/inventory.md"
REPORT = ROOT / "docs/environment/observed-state.json"   # gitignored

STATUS = {"present", "ported", "native", "not-applicable", "claude-only", "gh-cli", "awaiting-auth"}
CLIS = ("claude", "codex", "agy")


def load_manifest(path: Path = MANIFEST) -> dict:
    with open(path, "rb") as fh:
        return tomllib.load(fh)


def _self_test() -> int:
    ok: list[tuple[str, bool]] = []
    m = load_manifest()
    ok.append(("manifest loads", isinstance(m.get("plugins"), list) and isinstance(m.get("mcp"), list)))
    errs = validate_manifest(m)
    ok.append(("manifest is schema-valid", errs == []))
    ok.append(("bad status rejected", validate_manifest(
        {"plugins": [{"name": "x", "source": "s", "version": "v",
                      "components": ["skills"], "claude": "BOGUS", "codex": "present",
                      "agy": "present", "portability": "p"}], "mcp": []}) != []))
    failed = [n for n, p in ok if not p]
    for n, p in ok:
        print(f"  {'ok' if p else 'FAIL'}  {n}")
    print(f"env_inventory self-test: {len(ok) - len(failed)}/{len(ok)} passed")
    return 1 if failed else 0


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--self-test", action="store_true")
    args = ap.parse_args()
    if args.self_test:
        return _self_test()
    ap.error("no action given")
    return 2


if __name__ == "__main__":
    sys.exit(main())

def validate_manifest(m: dict) -> list[str]:
    """Return a list of human-readable schema errors (empty = valid)."""
    errs: list[str] = []
    plugins = m.get("plugins")
    mcp = m.get("mcp")
    if not isinstance(plugins, list):
        errs.append("top-level [[plugins]] must be a list")
        plugins = []
    if not isinstance(mcp, list):
        errs.append("top-level [[mcp]] must be a list")
        mcp = []
    names_seen: set[str] = set()
    for p in plugins:
        n = p.get("name", "<unnamed>")
        if n in names_seen:
            errs.append(f"duplicate plugin name: {n}")
        names_seen.add(n)
        for f in ("name", "source", "version", "components", "portability"):
            if f not in p:
                errs.append(f"plugin {n}: missing field '{f}'")
        if not isinstance(p.get("components"), list):
            errs.append(f"plugin {n}: components must be a list")
        for cli in CLIS:
            if p.get(cli) not in STATUS:
                errs.append(f"plugin {n}: {cli} status '{p.get(cli)}' not in {sorted(STATUS)}")
    for s in mcp:
        n = s.get("name", "<unnamed>")
        for f in ("name", "transport", "owner", "secret"):
            if f not in s:
                errs.append(f"mcp {n}: missing field '{f}'")
        if s.get("owner") not in ("reconcile", "bootstrap"):
            errs.append(f"mcp {n}: owner must be reconcile|bootstrap")
        for cli in CLIS:
            if s.get(cli) not in STATUS:
                errs.append(f"mcp {n}: {cli} status '{s.get(cli)}' not in {sorted(STATUS)}")
    return errs
```

REDACTED = "<redacted>"
# key names whose VALUES are always secret-bearing
_SECRET_KEY = re.compile(r"(token|secret|key|cred|password|passwd|cookie|auth|session|bearer)", re.I)
# env vars that are tuning/config, not secrets (values kept)
_SAFE_ENV = {"UV_HTTP_TIMEOUT"}


def _redact_url(u: str) -> str:
    try:
        parts = urlsplit(u)
    except ValueError:
        return REDACTED
    netloc = parts.netloc
    if "@" in netloc:                       # strip userinfo
        netloc = REDACTED + "@" + netloc.split("@", 1)[1]
    query = REDACTED if parts.query else ""  # drop query values wholesale
    return urlunsplit((parts.scheme, netloc, parts.path, query, ""))


def sanitize(obj):
    """Recursively strip every value-bearing channel; keep only safe symbolic refs."""
    if isinstance(obj, dict):
        out = {}
        for k, v in obj.items():
            if isinstance(v, str) and _SECRET_KEY.search(str(k)) and k not in _SAFE_ENV:
                out[k] = REDACTED
            elif str(k).lower() in ("env", "headers", "cookies"):
                out[k] = _sanitize_map(v)
            elif isinstance(v, str) and "://" in v:
                out[k] = _redact_url(v)
            else:
                out[k] = sanitize(v)
        return out
    if isinstance(obj, list):
        return [sanitize(x) for x in obj]
    return obj


def _sanitize_map(v):
    """env/headers/cookies: keep names, redact values unless explicitly safe."""
    if not isinstance(v, dict):
        return sanitize(v)
    return {k: (v[k] if k in _SAFE_ENV else REDACTED) for k in v}
```

PROBE_ERROR = object()   # sentinel: source could not be observed


def _run(cmd: list[str], timeout: int = 20) -> str | None:
    """Non-interactive, time-bounded. Returns stdout, or None on any failure."""
    env = {**os.environ, "TERM": "dumb", "PAGER": "cat", "NO_COLOR": "1"}
    try:
        r = subprocess.run(cmd, capture_output=True, text=True, timeout=timeout, env=env)
    except (FileNotFoundError, subprocess.TimeoutExpired):
        return None
    return r.stdout if r.returncode == 0 else None


def _parse_json_list(text: str):
    """Parse a JSON array/obj of entries → list; PROBE_ERROR on unusable input."""
    if not text or not text.strip():
        return PROBE_ERROR
    try:
        data = json.loads(text)
    except json.JSONDecodeError:
        return PROBE_ERROR

        for key in ("mcpServers", "servers", "plugins"):
            if key in data:

                return list(v) if isinstance(v, dict) else v
        return list(data.keys())
    return data if isinstance(data, list) else PROBE_ERROR

```python
def merge(existing: dict, additions: dict) -> dict:
    if not isinstance(existing, dict):
        raise ValueError("existing config must be a JSON object")
    out = json.loads(json.dumps(existing))          # deep copy, preserves unknown keys
    dst = out.setdefault(SERVERS_KEY, {})
    for name, spec in additions.get(SERVERS_KEY, {}).items():
        if name in dst:
            if dst[name] != spec:
                raise MergeConflict(
                    f"server '{name}' already exists with a different definition; refusing to clobber")
            continue                                 # identical → idempotent no-op
        dst[name] = spec
    return out


def write_merged(path: Path, additions: dict) -> dict:
    existing = json.loads(path.read_text()) if path.exists() else {}
    merged = merge(existing, additions)
    fd, tmp = tempfile.mkstemp(dir=str(path.parent), suffix=".tmp")
    try:
        with os.fdopen(fd, "w") as fh:
            json.dump(merged, fh, indent=2)
            fh.write("\n")
        os.replace(tmp, path)                        # atomic
    finally:
        if os.path.exists(tmp):
            os.unlink(tmp)
    return merged
```

def secrets_doc_gaps(refs: set[str], documented: set[str], allow: set[str]) -> dict:
