## Deferred live-config recommendations (apply yourself)

These configuration actions cannot be auto-applied by the reconcile engine and are recommended for manual tuning:

1. **Leverage G1 Credits Fallback:**
   If you have a G1 credit allocation, open `/settings` or `/config` inside agy and toggle the **UseG1Credits** option to `true` (or add `"UseG1Credits": true` in `~/.gemini/antigravity-cli/settings.json`) to allow automatic fallback when standard quota is exhausted.
2. **Utilize `proceed-in-sandbox` Permission Mode:**
   To maximize velocity without compromising safety, select **Proceed-in-sandbox** in the terminal execution policy menu (under `/config`), which auto-approves sandboxed commands while retaining manual approval prompts for host filesystem bypasses.
3. **Hide Personal Account Details in TUI Headers:**
   To keep your terminal clean, or for privacy when screen sharing/streaming, add the following to your shell profile (e.g., `~/.bashrc`):
   ```bash
   export AGY_CLI_HIDE_ACCOUNT_INFO=true
   ```
4. **Use /credits, /usage, and /quota in Real-Time:**
   Make use of these new slash commands in agy TUI to inspect your quota consumption and credit balance immediately.
